## Booking Management API

### Описание
API для управления бронированием автомобилей.

### Эндпоинты

#### 1. Создать бронирование
**POST** `/bookings`
- **Описание:** Создаёт новое бронирование

#### 2. Обновить бронирование
**PUT** `/bookings/{id}`
- **Описание:** Обновляет данные бронирования

#### 3. Отменить бронирование
**DELETE** `/bookings/{id}`
- **Описание:** Отменяет бронирование

#### 4. Получить стоимость бронирования
**GET** `/bookings/{id}/price`
- **Описание:** Возвращает стоимость бронирования

#### 5. Получить бронирования пользователя
**GET** `/bookings/user/{userId}`
- **Описание:** Возвращает бронирования указанного пользователя

#### 6. Подтвердить бронирование
**PUT** `/bookings/{bookingId}/confirm`
- **Описание:** Подтверждает бронирование администратором

#### 7. Получить все бронирования
**GET** `/bookings`
- **Описание:** Возвращает список всех бронирований
