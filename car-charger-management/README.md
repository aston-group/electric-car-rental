## Charger Management API

### Описание
API для управления зарядными станциями.

### Эндпоинты

#### 1. Получить все зарядные станции
**GET** `/chargers`
- **Описание:** Возвращает список всех зарядных станций

#### 2. Получить зарядную станцию по ID
**GET** `/chargers/{id}`
- **Описание:** Возвращает зарядную станцию по ID

#### 3. Создать новую зарядную станцию
**POST** `/chargers/create`
- **Описание:** Создаёт новую зарядную станцию

#### 4. Обновить данные зарядной станции по ID
**PUT** `/chargers/{id}`
- **Описание:** Обновляет данные зарядной станции по ID

#### 5. Удалить зарядную станцию по ID
**DELETE** `/chargers/{id}`
- **Описание:** Удаляет зарядную станцию по ID

#### 6. Занять свободную зарядную станцию по ID
**GET** `/chargers/{id}/start?carId={param}`
- **Описание:** Помечает зарядную станцию как использующуюся на данный момент по ID

#### 7. Освободить использующуюся зарядную станцию по ID
**GET** `/chargers/{id}/end?carId={param}`
- **Описание:** Помечает зарядную станцию как доступную для использования по ID
