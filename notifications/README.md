## Notifications API

### Описание
API для управления уведомлениями.

### Эндпоинты

#### 1. Получить все уведомления
**GET** `/notifications`
- **Описание:** Возвращает список всех уведомлений
- **Коды ответа:**
    - `202 Accepted` - Успешно получен список уведомлений

#### 2. Получить уведомление по Id
**GET** `/notifications/{notificationId}`
- **Описание:** Возвращает уведомление с указанным Id
- **Коды ответа:**
    - `202 Accepted` - Уведомление найдено

#### 3. Получить уведомления по статусу
**GET** `/notifications/status`
- **Описание:** Возвращает список уведомлений с указанным статусом
- **Параметры запроса:**
    - `status` (по умолчанию: SUCCESS) - статус уведомления
- **Коды ответа:**
    - `202 Accepted` - Успешно получен список уведомлений

#### 4. Изменить статус уведомления
**PATCH** `/notifications`
- **Описание:** Обновляет статус указанного уведомления
- **Коды ответа:**
    - `200 OK` - Статус уведомления успешно обновлён

#### 5. Создать новое уведомление
**POST** `/notifications`
- **Описание:** Создаёт новое уведомление
- **Коды ответа:**
    - `201 Created` - Уведомление успешно создано

#### 6. Удалить уведомление по Id
**DELETE** `/notifications/{notificationId}`
- **Описание:** Удаляет уведомление с указанным Id
