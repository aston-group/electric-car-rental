## Car Management API

### Описание
API для управления автомобилями.

### Эндпоинты

#### 1. Создать новый автомобиль
**POST** `/cars/create`
- **Описание:** Создаёт новый автомобиль
- **Коды ответа:**
    - `200 OK` - Автомобиль успешно создан

#### 2. Получить автомобиль по ID
**GET** `/cars/get/{id}`
- **Описание:** Возвращает автомобиль по указанному ID
- **Коды ответа:**
    - `200 OK` - Автомобиль найден
    - `404 Not Found` - Автомобиль не найден

#### 3. Получить список всех автомобилей
**GET** `/cars/all`
- **Описание:** Возвращает список всех автомобилей
- **Коды ответа:**
    - `200 OK` - Успешно получен список автомобилей

#### 4. Обновить данные автомобиля по ID
**PUT** `/cars/update/{id}`
- **Описание:** Обновляет данные автомобиля по ID
- **Коды ответа:**
    - `200 OK` - Автомобиль успешно обновлён
    - `404 Not Found` - Автомобиль не найден

#### 5. Удалить автомобиль по ID
**DELETE** `/cars/del/{id}`
- **Описание:** Удаляет автомобиль по ID
- **Коды ответа:**
    - `204 No Content` - Автомобиль успешно удалён
